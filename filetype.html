Interactive Version 0.1
By Savestate

Points: 4

Selections:


You happen across an unusual silver flash drive, simply lying on the street. Curiousity overrides caution, and you check the contents. It holds a series of strange, small programs, with rather minimalistic explanations; only a few sentences, each. Supposedly, each works by converting ordinary files into a new, anomalous filetype. It is then that the flash drive starts to simply dissolve. In the short time before it becomes completely nonfunctional, you are able to extract only four of these unusual programs.


Though clearly supernatural, the programs you retrieve are still programs. They can be copied freely, transferred between computers systems, used by others, and integrated into other software. They cannot, however, be modified, nor can anything be learned from observing their inner works; each converter is effectively a black box.


Accepts: Any image file.

Function: Converts the source into a ten second audioless video, displaying the moments before and after the moment of the image. May function oddly on unreal images.


Accepts: Any image file.

Function: Converts the source into an infinite-resolution replica. Automatically handles conflicts with programs that measure based on set numbers of pixels. At high zoom, may adjust image to be as if it was taken at the zoomed-in location.


Accepts: Any image file.

Function: Converts the source into a panorama. User can rotate view 360 degrees in every direction, observing the are around where the image was originally taken. May function oddly on unreal images.


Accepts: Any source code.

Function: Converts the source into a compiled program that will, at minimum, run, and on any computer system. Automatically resolves typos, logic errors, inappropriate constants, and similar based on the intent of the program's original writer(s).


Accepts: Any compiled program or .exe file.

Function: Converts the source into a superior format, capable of running on any computer. Improves run speed to varying degrees. Games will always run at maximum FPS, and never run out of memory. Purely practical programs will run a maximum of 100x as quickly.


Accepts: Any file or folder.

Function: Converts the source into a losslessly compressed file, which will take up precisely 99 kilobytes of memory.


Accepts: Any imagined sound or image.

Function: Generates a digital anchor to the original fantasy. When displayed by any means, sapient beings will perceive the original imagined idea.


Accepts: Any audio file.

Function: Converts the source into a perfect-quality replica. Physical limitations of audio emitters are disregarded when playing this file. Individual sound sources from the original recording can be dialed up or down in volume and clarity.


Accepts: Any image file, or any text file.

Function: Adds text to image files, providing story elements in line with the user's vague desires. Adds imaging and illustration to text files, enhancing them in line the user's vague desires.


Accepts: Any video file.

Function: Converts the source into an "as directed by George Lucas" version. One time only, can reset this to any other director. This reset propogates through all copies of the program.


Accepts: Any text file.

Function: Converts the source into an "as written by JK Rowling" version. One time only, can reset this to any other published author. This reset propogates through all copies of the program.


Accepts: Any audio or video file.

Function: Converts the source into an "as played by Weird Al" version. One time only, can reset this to any other musician. This reset propogates through all copies of the program.
